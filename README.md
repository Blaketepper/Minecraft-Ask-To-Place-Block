# Minecraft Ask To Place Block
This is a Minecraft mod where every time you place a block it tells you if you want to place it there.
Getting this setup up is easy if you follow these easy steps.

**I am assuming that you already have Neoforge 1.21.8 installed!**
1. You need to put in the downloaded mod into `C:\Users\<Username>\AppData\Roaming\.minecraft\Mods`.
2. Go to the lanucher and open Minecraft NeoForge 1.21.8
3. When you open into a world you have 1 step run this command `/gamerule asktoplaceBlocks true` than hit enter.
4. You are good to go!
